Privacy Policy

Effective date: January 20, 2026

StraightShot Auto analyzes vehicle listing pages on Facebook Marketplace and provides an on-page overlay with extracted details and AI-generated insights.

Data We Process

  • Listing page content (title, price, mileage, description, specs)
  • Listing URL for the current page
  • Network metadata handled by hosting providers for security and rate limiting

How We Use Data

  • Generate analysis and insights for the current listing
  • Improve reliability (rate limiting and caching)

Data Sharing

Listing data is sent to our backend service and processed by OpenAI to generate analysis. We do not sell user data.

Data Retention

Analysis responses may be cached briefly to reduce repeated requests. We do not store user accounts or personal identifiers.

Your Choices

You can disable or uninstall the extension at any time. The extension runs only on Facebook Marketplace item pages.
